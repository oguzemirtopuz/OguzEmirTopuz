## 🚀 Featured Projects

### 🧠 [JARVIS Cognitive OS](https://github.com/oguzemirtopuz/JARVIS-Cognitive-OS)
> *An autonomous, self-healing AI operating system that plans, executes, and recovers from failures on its own.*

The most architecturally sophisticated project in this portfolio. JARVIS is a full cognitive agent framework — not a chatbot wrapper. It features:
- **39 specialized core modules** spanning planning, memory, perception, and execution
- **Episodic Vector Memory** — recalls past successes and failures using ChromaDB embeddings
- **Reflection Engine** — rewrites its own plan when it encounters errors, with zero human intervention
- **Dynamic Skill Synthesizer** — generates new Python tools on the fly using the LLM, validated through AST sandbox before hot-loading
- **Semantic Router** — routes simple commands locally in milliseconds; only calls the LLM for complex cognition
- **Multi-input** — voice (Whisper STT), GUI, and CLI all feeding the same async orchestration core

**[⭐ Star JARVIS](https://github.com/oguzemirtopuz/JARVIS-Cognitive-OS)**

### 📊 [YouTube Analyse Pro — SaaS Edition](https://github.com/oguzemirtopuz/YouTube-Analyse-Pro-SaaS-Edition)
> *Mathematically reverse-engineer why videos go viral. Then replicate it.*

A dual-component platform (Desktop App + Chrome Extension) for content creators who want data-driven channel growth:

- **BabaClutch Chaos Metric** — custom NLP algorithm measuring psychological engagement density
- **DNA Scoring Engine** — weighted virality formula (Hook 40% + Tempo 40% + CTA 10% + Emotion 10%)
- **AES-encrypted multi-user SaaS** with Google OAuth2 and PDF report generation

**[⭐ Star YouTube Analyse Pro](https://github.com/oguzemirtopuz/YouTube-Analyse-Pro-SaaS-Edition)**

### 🔬 [PyAuditor](https://github.com/oguzemirtopuz/PyAuditor)
> *25 static analysis rules + AI-powered logic review. No pip install. No API keys. Just run it.*

The fastest way to audit any Python project before shipping:
- **25-rule AST engine** — catches asyncio races, silent exception swallows, mutable default args, shadow builtins, and more
- **BYOAI (Bring Your Own AI)** — generates an optimized prompt that you paste into ChatGPT / Claude / Gemini to detect logical errors no static rule can find
- **Zero dependencies** — runs on pure Python stdlib
- **Sub-2-second scans** on thousands of lines of code

**[⭐ Star PyAuditor](https://github.com/oguzemirtopuz/PyAuditor)**
